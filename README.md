# Tooltopia

A collection of web-based utilities and interactive tools for various tasks and visualizations.

## Features

- **CSC Formatter** - Format and beautify CSC source code
- **Text Grep** - Batch text search tool
- **Excel Grep** - Multi-column table search tool
- **Linear Algebra** - Interactive visualization for linear algebra concepts
- **Linear Transformation** - Visualize linear transformations
- **Weyl Sequence** - Interactive visualization of Weyl sequence trajectories
- **Lights Out** - Classic puzzle game with multiple variants
- **SRT2TXT** - Convert SRT subtitle files to plain text
- **Calculator** - Machine learning calculator with matrix operations

## Usage

Simply open `index.html` in a web browser to access the tool collection. All tools run entirely in the browser - no installation or backend required.

### Direct Tool Access

Each utility can be accessed directly via its HTML file:
- CSC Formatter: `utils/CSCSourceFormatter/index.html`
- Text Grep: `utils/TextGrep/index.html`
- Excel Grep: `utils/ExcelGrep/index.html`
- Linear Algebra: `utils/LinearAlgebra/index.html`
- Linear Transformation: `utils/LinearTransformation/index.html`
- Weyl Sequence: `utils/WeylSequence/index.html`
- Lights Out: `utils/LightsOut/index.html`
- SRT2TXT: `utils/SRT2TXT/index.html`
- Calculator: `utils/MachineLearningCalculator/index.html`

## Deployment

### GitHub Pages
1. Enable GitHub Pages in repository settings
2. Select the main branch as source
3. Access via `https://<username>.github.io/<repository>`

### Local Deployment
Simply open `index.html` with any modern web browser (Chrome, Firefox, Edge, Safari).

## Browser Compatibility

Works on modern browsers including:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

Mobile and tablet devices are fully supported with responsive design.
