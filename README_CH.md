# Tooltopia

网页版实用工具和交互式可视化工具集合。

## 功能特性

- **CSC 格式化器** - 格式化和美化 CSC 源代码
- **文本搜索** - 批量文本搜索工具
- **表格搜索** - 多列表格搜索工具
- **线性代数** - 线性代数概念交互式可视化
- **线性变换** - 线性变换可视化
- **Weyl 序列** - Weyl 序列轨迹交互式可视化
- **点灯游戏** - 经典益智游戏及多种变体
- **字幕转换** - 将 SRT 字幕文件转换为纯文本
- **计算器** - 机器学习计算器，支持矩阵运算

## 使用方法

在浏览器中打开 `index.html` 即可访问工具集合。所有工具完全在浏览器中运行，无需安装或后端支持。

### 直接访问工具

每个工具可通过其 HTML 文件直接访问:
- CSC 格式化器: `utils/CSCSourceFormatter/index.html`
- 文本搜索: `utils/TextGrep/index.html`
- 表格搜索: `utils/ExcelGrep/index.html`
- 线性代数: `utils/LinearAlgebra/index.html`
- 线性变换: `utils/LinearTransformation/index.html`
- Weyl 序列: `utils/WeylSequence/index.html`
- 点灯游戏: `utils/LightsOut/index.html`
- 字幕转换: `utils/SRT2TXT/index.html`
- 计算器: `utils/MachineLearningCalculator/index.html`

## 部署方式

### GitHub Pages 部署
1. 在仓库设置中启用 GitHub Pages
2. 选择 main 分支作为源
3. 通过 `https://<用户名>.github.io/<仓库名>` 访问

### 本地部署
使用任何现代浏览器（Chrome、Firefox、Edge、Safari）直接打开 `index.html` 即可。

## 浏览器兼容性

支持所有现代浏览器:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

完全支持移动设备和平板电脑，采用响应式设计。
