Project Scaffolder

项目结构初始化与逆向生成工具。支持“目录树文本 ⇄ 压缩包”互转，并自动管理文件头注释。

功能模式

📦

点击选择或拖拽 ZIP 压缩包

支持包含项目代码的 .zip 文件

生成结果

就绪

配置好左侧结构后点击下方按钮生成

使用说明

Text → Zip (生成模式)

1. 粘贴带有缩进或 `├──` 符号的目录树文本。
2. 支持在行尾使用 `#` 添加注释，这将被提取为文件的 Description。
3. 勾选"生成文件头注释"，工具会自动根据文件类型生成规范的 Header。

Zip → Text (解析模式)

1. 上传项目的 ZIP 压缩包。
2. 工具会递归扫描目录结构。
3. 如果勾选提取功能，会自动读取代码文件的头部注释，将 Description 字段追加到目录树中。
