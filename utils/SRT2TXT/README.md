# SRT 转 TXT 工具

一个用于将 SRT 字幕文件转换为纯文本格式的 Web 应用程序。

## 功能特性

### 🎯 核心功能
- **SRT 文件解析**：自动解析 SRT 字幕文件格式
- **多种输入方式**：支持文件上传和直接粘贴内容
- **灵活的输出格式**：支持带时间戳和不带时间戳两种格式
- **智能分割选项**：不带时间戳时支持换行分隔和标点分隔
- **便捷的输出方式**：支持复制到剪贴板和下载为 TXT 文件

### 🆕 格式选项
- **带时间戳格式**：保留时间码信息，格式为 `[时间戳] 字幕文本`
- **不带时间戳格式**：仅提取纯文本内容
  - **换行分隔**：每段字幕独立一行
  - **标点分隔**：使用中文逗号连接所有字幕，形成连续文本

### 🎨 界面特性
- **响应式设计**：支持不同屏幕尺寸
- **现代化 UI**：渐变背景和毛玻璃效果
- **流畅交互**：实时转换和格式切换
- **中文界面**：完整的中文用户界面
- **移动端优化**：触摸友好的交互设计

## 技术实现

### 前端技术
- **原生 JavaScript**：核心解析和转换逻辑
- **CSS3**：样式和动画效果
- **响应式设计**：适配移动端和桌面端
- **文件 API**：支持文件读取和下载
- **剪贴板 API**：支持一键复制

### 核心功能
- **SRT 解析**：智能识别序号、时间码和文本内容
- **格式转换**：灵活的输出格式配置
- **数据管理**：保存原始解析数据，支持实时格式切换

## 使用方法

### 输入方式

#### 1. 文件上传
- 点击"选择文件"标签
- 点击上传区域或直接拖拽 SRT 文件到上传区域
- 支持 `.srt` 格式文件

#### 2. 粘贴内容
- 点击"粘贴内容"标签
- 在文本框中粘贴 SRT 文件内容
- 支持实时转换

### 输出格式

#### 带时间戳格式
- 格式：`[00:00:01,000 --> 00:00:04,000] 字幕文本`
- 每段字幕独立一行
- 保留完整的时间码信息

#### 不带时间戳格式

**换行分隔**
- 每段字幕独立一行
- 保持原始段落结构
- 示例：
```
第一段字幕文本
第二段字幕文本
第三段字幕文本
```

**标点分隔**
- 使用中文逗号连接所有字幕
- 形成连续的文本流
- 示例：
```
第一段字幕文本，第二段字幕文本，第三段字幕文本
```

### 输出操作

#### 复制到剪贴板
- 点击"复制到剪贴板"按钮
- 转换结果将复制到系统剪贴板
- 可直接粘贴到其他应用程序

#### 下载 TXT 文件
- 点击"下载 TXT 文件"按钮
- 自动下载为 `subtitle.txt` 文件
- 可在本地保存和编辑

#### 清空
- 点击"清空"按钮
- 清除所有输入和输出内容
- 重置所有选项

## SRT 文件格式

SRT (SubRip Subtitle) 是一种常见的字幕文件格式，结构如下：

```
1
00:00:01,000 --> 00:00:04,000
第一段字幕文本

2
00:00:05,000 --> 00:00:08,000
第二段字幕文本
```

每段字幕包含：
- **序号**：字幕段的编号
- **时间码**：开始时间和结束时间
- **文本内容**：字幕文本（可多行）

## 转换规则

### 带时间戳格式
- 保留时间码信息
- 格式：`[时间戳] 文本内容`
- 每段字幕独立一行

### 不带时间戳格式

#### 换行分隔
- 去除序号和时间码
- 保留文本内容
- 每段字幕独立一行
- 保持原始段落结构

#### 标点分隔
- 去除序号和时间码
- 保留文本内容
- 使用中文逗号连接
- 形成连续文本流
- 去除所有换行符
